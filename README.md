# Clock-In-Server
## 功能描述

学生打卡信息的快速统计。

## 使用指南

点击【修改名单】按钮可以修改打卡名单，修改完之后记得进行保存。

在保存好名单之后即可输入已签到的人员名单，点击【查询】按钮便可得到未签到的人员名单。
